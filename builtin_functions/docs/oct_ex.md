### Documentation:

[oct](https://docs.python.org/ja/3/library/functions.html#oct)  
整数を8進数の文字列に変換

### Syntax:

```oct(num)```

- **num** (int): 8進数に変換する整数

### Returns:

str

### Example: 

```python
>>> oct(30)
'0o36'

>>> format(30, '#o')
'0o36'

>>> format(30, 'o')
'36'

>>> format(30, '#o')
'0o36'

```
