### Documentation:

[sum](https://docs.python.org/ja/3/library/functions.html#sum)  
数値の和

### Syntax:

```sum(a[, b])```

- **a** (tuple): int/float の配列
- **b** (int/float, optional): tuple に対して足す数値

### Returns:

int/float

### Example: 

```python
# returns the sum of a
>>> a = (1, 2, 3.5)
>>> x = sum(a)
6.5

# returns the addition of sum(a) and b 
>>> a = (1, 2, 3)
>>> x = sum(a, 4)
10

```
